# 心愿社角色卡

“心愿社”是一张面向 SillyTavern 的现代都市互动角色卡，包含 Vue 前端状态栏、MVU 变量系统、任务大厅、论坛、商城、人物状态与衣物/道具管理。

## 使用要求

- SillyTavern
- 酒馆助手插件及 MVU 支持
- 可访问 `testingcf.jsdelivr.net` 的网络环境

## 安装

1. 下载 [`cards/心愿社/心愿社.json`](cards/心愿社/心愿社.json)。
2. 在 SillyTavern 中导入角色卡。
3. 开启酒馆助手与 MVU 功能后开始新对话。

状态栏正式地址：

`https://testingcf.jsdelivr.net/gh/yjlovegy/xinyuanshe-card/dist/心愿社/界面/状态栏/index.html`

## 目录

- `cards/心愿社/`：角色卡源文件与可导入 JSON
- `src/心愿社/`：Vue 状态栏源码
- `dist/心愿社/`：CDN 使用的生产构建

## 内容说明

本项目包含成人题材设定。所有相关角色和平台用户均被设定为成年人；互动应建立在自愿、知情与可撤回同意的基础上。
